# incue-website

Incue Inc. 홈페이지 Cloudflare Pages 정적 사이트입니다.

## 배포 설정

- Framework preset: None
- Build command: 비움
- Build output directory: `/` 또는 repository root
- Production branch: `main`

## 로컬 검증

```bash
python -m http.server 4173
```
